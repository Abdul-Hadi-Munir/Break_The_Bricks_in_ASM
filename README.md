# Brick-BreakerX86
## Description 
The Brick game in Assembly language as my COAL project.
The game includes:
1. A start menu with instructions
2. Multiple difficulty levels
3. Vibrant 4Ã—10 brick layout
4. Smooth paddle control & ball movement

## Environment 
Assembler: &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; NASM<br/>
DOSBox x86 emulator

#### Create executable
```
nasm game.asm -o game.com
```
#### Run executable
```
game.com
```
